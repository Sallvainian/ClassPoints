_Generated 2026-07-17 (exhaustive full rescan; HEAD `e34bbf3` on `main`)._

ClassPoints is a single-part monolith — one React SPA backed by Supabase, which since PR #132/#136 ALSO ships as a Capacitor 8 native app (iOS + Android). The repo's interesting structure lives in `src/`, `supabase/`, `tests/`, `scripts/`, and `.github/workflows/`; `ios/` and `android/` are standard generated Capacitor platform projects.

Naming note: `001`–`012` are hand-authored with a zero-padded `0NN` prefix; the five timestamp-prefixed files were emitted by `supabase migration new`. Files apply in lexicographic order, so the zero-padded set sorts before the timestamped ones. New migrations should be created with `supabase migration new <name>` (timestamp-prefixed; sorts after the legacy set) — do not reuse or renumber a zero-padded prefix.

## Critical entry points

| Path                          | What it is                                                                                                                                                                                      |
| ----------------------------- | ----------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| `src/main.tsx`                | App bootstrap. Self-hosted `@fontsource` imports; mounts `<App />` inside `QueryClientProvider`; registers DevtoolsGate (DEV-only).                                                             |
| `src/App.tsx`                 | Provider stack + view router (home / dashboard / settings / migration / profile). View persists via `localStorage:app:view`. Native glue: `hideSplash()` mount effect + `registerBackButton()`. |
| `src/contexts/AppContext.tsx` | `useApp()` — UI/session state ONLY (activeClassroomId + setActiveClassroom, 33 LOC; Phase 4 dissolved the facade). New components MUST call hooks directly.                                     |
| `src/lib/supabase.ts`         | Single Supabase client (Preferences auth storage on native). `unwrap()` / `isPostgrestError` / `isNetworkClassAuthError`.                                                                       |
| `src/lib/queryClient.ts`      | TanStack QueryClient with ADR-005 defaults (load-bearing for the optimistic-mutation Phases 1–3).                                                                                               |
| `src/lib/queryKeys.ts`        | Single source of truth for query keys — used for both reads AND invalidations to keep them in sync.                                                                                             |
| `capacitor.config.ts`         | Native shell config: appId, `webDir: 'dist'`, SplashScreen (`launchAutoHide: false`), `CAP_SERVER_URL` live-reload escape hatch.                                                                |
| `scripts/dev.mjs`             | `npm run dev` entrypoint. Local-by-default; starts/stops local Supabase + preflights Docker.                                                                                                    |
| `playwright.config.ts`        | E2E. Fail-closed network allow-list (loopback, RFC1918, Tailscale CGNAT). 4 projects: setup / chromium / mobile (390×844) / ipad (834×1194).                                                    |
| `supabase/migrations/`        | 17 migrations (12 zero-padded `0NN` + 5 timestamp-prefixed CLI files); applied in lexicographic order.                                                                                          |
| `supabase/functions/`         | `delete-account` — the sole Edge Function (JWT-derived identity; `verify_jwt = true` in config.toml).                                                                                           |
