Naming note: `001`‚Äì`012` are hand-authored with a zero-padded `0NN` prefix; the single timestamp-prefixed file was emitted by `supabase migration` during the PR #86 linter-hardening work. Files apply in lexicographic order, so the zero-padded set sorts before the timestamped one. New migrations should be created with `supabase migration new <name>` (timestamp-prefixed; sorts after the legacy set) ‚Äî do not reuse or renumber a zero-padded prefix.

## Critical entry points

| Path                          | What it is                                                                                                                                                  |
| ----------------------------- | ----------------------------------------------------------------------------------------------------------------------------------------------------------- |
| `src/main.tsx`                | App bootstrap. Mounts `<App />` inside `QueryClientProvider`; registers DevtoolsGate (DEV-only).                                                            |
| `src/App.tsx`                 | Provider stack + view router (home / dashboard / settings / migration / profile). View persists via `localStorage:app:view`.                                |
